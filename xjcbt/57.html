测试了 24 款局域网传文件 App，终于找到适合我的了！

晓皎 2020-02-16 18:58:30 2,322 本文共2452个字，预计阅读需要7分钟。

前言

此文本来在两个月前就已经准备好一半，App 也测试得差不多了，就是一直没有写文整理，如今终于补齐。

想必大家都应该遇到过这种情况：想用手机给室友传一部电影（或者室友分享给你一部电影），用 QQ/微信且先不说他们支持传几个 G 的文件，即便能传速度不仅慢，而且浪费流量。

你会问为什么不用 QQ 面对面快传？这首先得双方已加好友，且手机上还有 QQ 的情况吧，我看不少人用微信了。况且遇到要给陌生人传文件怎么办？

总之别杠，您觉得您有局域网传文件的需求就看。

市面上有很多传文件 App，然而不少 App 都需要双方都安装他们的 App 才能用。你想想你传一个文件给别人还要别人也装个同样的 App，对方不得口吐芬芳？

本着不麻烦别人的方针，我们需要用一个 App 就完成分享的任务。当然这个 App 不仅能分享文件，还能接收别人传来的文件那就更好了。意思是双方仅需一人安装 App，即可实现双方的文件分享与接收，岂不美哉？

经过多方对比，我测试筛选了在谷歌应用商店和酷安上的 24+ 款相关 App，发现真的有这样的 App，双方通过浏览器进行传文件，具体请看下面的详细描述。

局域网传文件温馨提示：

局域网传文件只要双方在同一网络、WiFi、热点情况下，都可以。其传输速度取决于路由器、网卡、手机 WiFi 模块等，这就是千兆路由器显威力的时候了。

优先连接同一网络、WiFi，在上述二者都没有的情况下才开热点传文件（为了防止意外偷跑流量，传输时可将数据流量关闭），因为手机 WiFi 模块性能不强，速度不快，一般就几 MB/s。

不合格 App 名单

下列 App 均需要双方安装相同 App 才能使用，没什么好说的，不多 BB。

  • YShare（v2.0）
  • ShareThis（v1.8.36）
  • Share all（v1.0.3）
  • X-Transfer（v3.2.2）
  • Swift File Transfer（v1.2.1）
  • 快牙 Google Play 版（v5.8.7）
  • 快牙精简版 Google Play 版（v1.6.3）
  • 飞鸽传书（v2.4）
  • 快传（v1.3.0）

也不是不能用 App 名单

下列 App 仅需一方安装 App 即可使用。可用浏览器分享/接受文件，但整体功能上不完善或需要付费使用，不推荐使用。

  • 超传（v2.3.0）（连接有问题）
  • 备份助手（v10.01）（手机互传貌似不行，电脑互传可以）
  • Portal（v1.3.2）（仅可接收，不可分享，必需扫码才能传文件，二维码刷新缓慢，国外团队制作）
  • WiFi File Transfer（v1.9.0）（可以分享，接收（接收文件限制 5MB 大小，购买 Pro 版可解锁），制作简陋）
  • Fast File Transfer（v2.1.5）（可以分享，不可接收，购买专业版可解锁接收功能）
  • FileShare（v2.0）（可以分享，接收，GitHub 开源，可在酷安下载，个人开发者，制作较简陋）

优秀 App 名单

文件闪传：支持投屏（使用浏览器投屏，无需安装软件）、剪贴板同步，没有花里胡哨的多余功能。有开屏广告，花 5 元购买高级版可解除（目前高级版的唯一功能是去广告）

互传：vivo 出品，支持投屏（需要额外安装软件），可搜索手机文件（太慢了），一次只能连接一个设备，有一键换机等其他功能，略显累赘。使用说明：点击「网页传」，对方在浏览器上输入局域网 IP 和端口即可。

飞鸟快传：手机端浏览器界面适配一般，仅可接收，不可分享，可开启浏览器「电脑模式」解决。使用说明：打开 App 之后，点击左上角打开菜单栏，选择「网页传输」，按 App 说明操作即可。

闪传：使用说明：打开 App 之后，点击左上角打开菜单栏，选择「连接电脑」，按 App 说明操作即可。

ShareMe：「传至电脑」是使用 FTP 连接电脑，可在电脑浏览器中打开，只能下载；若在资源管理器中打开，可上传/下载。一般选择「网页传」即可，必需开热点才能连接。

局域网精灵：使用说明：点击有下角「+」图标，选择想要传输的文件，左上角可查看局域网 IP 和端口，对方在浏览器上输入局域网 IP 和端口即可。

网页分享：支持在线查看图片，播放视频。一次分享多张图片，可能有 Bug，不建议使用。

爱莫助手：支持在线查看图片，播放音乐，播放视频（支持 x264 编码，不支持 x265 编码），支持投屏（镜像）（使用浏览器投屏，无需安装软件）。甚至可以管理手机联系人、短信等，如果不需要这些功能，可在应用管理中关闭这些权限。使用说明：建议点击 App 主界面右上角，查看 IP 地址，对方输入IP地址即可。如果输入 web.airmore.cn 需要联网才能打开，输入 IP 地址可以断网使用。

AirDroid：整体上和爱莫助手非常相似，不过投屏需要下载软件。使用说明：和爱莫助手一样，建议输入 IP 地址使用，而非 web.airdroid.com，可在「AirDroid 网页端」查看 IP 地址，在设置中可修改端口号，随便选一个好记的就行。

你该作何选择？

远程删除文件：双方进行连接之后，未安装 App 那方可通过浏览器删除已装 App 一方手机里的文件。

移动端适配情况：对移动端浏览器界面是否有专门的优化。如果没有也不是不能用，就像是用手机来看 PC 端的桌面一样，可以放大缩小，不过有点费眼神罢了。

你需要想清楚是否需要「远程删除文件」这个功能，毕竟连接上你手机的人可以任意删除你的手机文件，一般人完全不需要这个功能，除非你经常对手机进行远程管理。

关于是否支持接收文件这点，我个人认为还是支持比较好，毕竟别人传文件给你比较方便。至于有没有不安好心的人传个病毒给你，同一个局域网下，相隔数米，他是什么样的人，大家心里还是有数吧。

综上所述，一般人选择局域网精灵即可，如果有远程管理手机的需要，可以选择爱莫助手/AirDroid，两款 App 各方面都差不多，支持安卓和 iOS。
