仅需一个聚合磁力、BT 种子资源导航站，解决你的所有找种子及看片需求

晓皎 2019-10-26 17:30:01 20,997 本文共1569个字，预计阅读需要4分钟。

更新（2020/02/24）：又找个一个类似聚 BT 网站叫 BTMOO：https://www.btmoo.com/

聚 BT 果然还是到了被禁都地步，请记住聚 BT 永久域名：https://jubtnet.github.io/jubt/

前言

大家好，我是晓皎。

之前我写过几篇文章来介绍国内外的 BT 种子网站和客户端，但说实话有些问题一直无法解决（问题在下列引文中有描述）。一次无意间在 GitHub 上看见一个导航网站的项目才发现此网站。真是缘，妙不可言。

本来是想私藏这个网站的，考虑再三还是决定写篇文章介绍一下。小声说一句：我写这篇文章的时候正是聚 BT 导航建站半周年，感觉很有可能被墙，或者……总之且用且珍惜吧！

最后一句话评价一下这个网站吧！它是我这么多年见到的唯一一个如此详细且专业的聚合磁力、BT 种子资源导航站。

聚 BT 导航简介

以下是站长的自述，没有兴趣的可跳过。

由于国内舆论及版权收紧，很多网站被关闭/墙掉，内容资源在网站、网盘上被下架，类似 115、迅雷这样的服务被监管阉割得基本残废，于是磁力/BT 搜索及下载又变成常用工具。

一直借助浏览器收藏夹、博客文章来维护最新可用的服务，但发现基本不可行： 1、意识使然。看大片对自己算是低频需求，虽然喜欢看，但平常忙，基本只有每周末才有闲暇，关键是潜意识中认为是非专业性工作，也不愿花时间去整理资源站

2、收藏夹收藏了一堆网址，但赶不上墙掉、关闭的速度，收藏夹里面大部分网址都不可用

3、博客文章大多写完后，后续也懒得更新维护；博客文章也很难成体系沉淀、完善。

也曾使用过一些提供此类服务的导航站，最终发现比收藏夹、博客文章形式维护更不靠谱。很多导航站初期很清爽，后来功能越做越复杂，更新维护也不频繁，而且基于利益考虑，推荐的很多服务基本很差。

于是乎，经常的套路就是：每一次用时，使用搜索引擎大法。但效果也不行，搜索出的质量都比较差。

一直未曾想过弄一个导航站来干什么，可能潜意识里觉得导航站是上世纪的东东，太 low。只不过确实不满意现有的收藏夹、博客文章、导航站形式，简单思考了一下解决方案，发现还是导航站形式最适合：

1、导航站信息架构本身就是结构化的，很适合此类需求场景；

2、维护一个导航站，算是一种承诺，可以强迫自己更成体系化形式收藏、整理相关资源，实际上也是节省自己时间。

于是故事的开局类似许多互联网的鸡汤故事：xx 不满意现有的 yy 服务，自己做了一个……

聚 BT 导航站：https://jubt.net

网站定位：聚合最优质的磁力、BT 资源。

导航站板块组成：

  • 磁力搜索：收录优秀的磁力/BT 搜索引擎
  • 磁力影视站：收录优秀的影视资源站
  • 在线影视站：收入优秀在线电影、电视资源站
  • 磁力APP：收录优秀的磁力/BT APP，只收录有明确渠道来源的 APP，不收录各种不明来源的 xx 神器。
  • 磁力软件：收录优秀的磁力/BT PC软件
  • 云盘搜索：收录优秀的网盘搜索引擎
  • 离线下载：收录优秀的离线下载服务
  • 浏览器扩展：收录优秀的老司机必备浏览器扩展
  • 油猴脚本：收录优秀的 Greasemonkey/Tampermonkey 脚本，老司机必备神器
  • 福利资源：网友投稿推荐的福利资源

难能可贵的是，自从 2019/04/26 建站以来，聚 BT 站长一直在更新，突然感觉我有点自惭形秽。

UNBLOCKED（国外）导航简介

类似的还有一个国外的导航网站，由于是国外的导航，所以里面收录的都是国外的网站，主要有 BT 种子网站板块、流媒体网站板块（即在线视频）、体育板块……

UNBLOCKED（国外）导航官网：https://unblocked.ltda

如果打不开，可用备用地址：https://unblocked.nz、https://unblocked.dk、https://unblocked.win
