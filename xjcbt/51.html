专题引言

这是我准备了很久的新专题系列——我们究竟该如何下载网上视频？

此专题主要的目的是找到下载国内视频网站最有效的方法，后续我们以爱优腾（即爱奇艺、优酷和腾讯视频，下同）为代表进行下载测试。

不知道从什么时候起，各大视频网站纷纷取消了视频下载按钮，即便能下载的也有不少限制，国内以爱优腾三大巨头为代表：

  • 爱奇艺专用格式：.qsv
  • 优酷专用格式：.kux
  • 腾讯视频专用格式：.qlv

这些视频格式只能通过他们自家的播放器才能播放（当然可以通过某些转码软件转为 .mp4 格式，但这不在本文的讨论范围内）。我还记得某些人在讲 PPT 需要播放视频的时候，手动安装腾讯视频客户端，再打开 U 盘里 .qlv 视频的窒息操作。

4 款视频下载开源项目简介

经过我多年的收集，找到了目前最受欢迎的几款视频下载开源项目，按 GitHub 网站上 star 数量排名（star 可理解为收藏，点赞）：

  • youtube-dl：https://github.com/ytdl-org/youtube-dl（star：55353）
  • you-get：https://github.com/soimort/you-get（star：26965）
  • annie：https://github.com/iawia002/annie（star：7663）
  • ykdl：https://github.com/zhangn1985/ykdl（star：711）
  • Lulu：https://github.com/iawia002/Lulu（star：701，已经停更，目前作者主要更新上述的 annie 项目）

ykdl 和 annie 对国内网站支持甚好；annie 作为新星，下载 B 站十分方便；ykdl 下载爱奇艺很好； youtube-dl 下载油管奇绝，具体查看其支持站点及说明。

毫不夸张的说，使用这 4 款视频下载开源项目(youtube-dl+you-get+annie+ykdl)下载 90%+ 的视频都没有问题。只是配置和使用过程中门槛略高，等你熟练使用之后就知道这是何等神器。

  • youtube-dl 支持网站列表（支持国内外 1100+ 视频网站）
  • you-get 支持网站列表
  • annie 支持网站列表
  • ykdl 支持网站列表（专注下载国内视频网站）

我们先来看看这 4 款视频下载开源项目究竟支持哪些网站，由于本文篇幅有限，全部列出需要大量的篇幅，具体情况可点击上方链接进行查看。我们就来看看 annie 支持哪些网站，如上图所示， annie 支持下载国内最难对付的爱优腾。

我们就拿在爱奇艺播出的《中国达人秀第6季》来实际测试一下，如上图所示，成功解析到了视频下载地址，支持选择下载的视频分辨率。

如上图所示，我选择了 1080P 分辨率进行下载，实际下载速度也差不多跑满了我家宽带。但是，某些小伙伴就注意到了，我并没有说究竟怎么操作。

这 4 款开源程序的使用方法和我们经常接触的 Windows 软件不同，他们没有图形化界面，依靠 CMD 命令行操作（如上图所示），这就是为什么我在前面写不适合电脑小白的原因，不过其实操作也不难，按照教程来，谁都会用。

这 4 款开源程序的使用方法都大同小异，具体使用教程可百度搜索「项目名称+使用教程」，如搜索：youtube-dl 使用教程，网上有很多，我不想赘述。

AYY-VideoDown 简介

AYY-VideoDown 集合了 annie/ykdl/youtube-dl/you-get 四大网络视频下载组件，不过仍然采用命令行的方式下载视频，不过相比原版已经简单了很多。其中，annie 默认安装可以直接使用，其余三个可以自行选择安装。

AYY-VideoDown 是用于网络视频下载的工具，理论上支持腾讯视频/优酷/爱奇艺/B站/A站/芒果TV/搜狐视频/斗鱼/秒拍/tumblr /油管/CCTV/CNTV/熊猫直播/虎牙/凤凰视频等国内外各种网站的视频。

本脚本适用于懒人和小白，所以有一些可能需要的参数你可以自己在 CMD 里输入；基本默认下载，没有什么参数可以设置；需要特殊服务，可以去项目主页查看，如果你看得懂的话。

使用说明

教学视频： https://www.bilibili.com/video/av59555953

无意间在 B 站搜到一个简单的使用教程，说得很清楚，直接看视频吧，肯定比看文字来得直观，视频由 B 站 UP 主 @辣椒解说z 录制（如果视频被删除了，文末的下载链接里有备份）。

注意事项与解决方式

  1. 暂停下载不会删除已经下载的部分缓存，记得打开相应文件夹删除不想下载的视频缓存。
  2. 移动位置后，cmder 需要进入 Cmder_mini 后才能使用，如果用 cmder 代替了系统 cmd，这个时候迁移了位置就会导致 cmd 打不开。 解决办法：进入 Cmder_mini，打开 Cmder.exe 然后再执行「安装与更新.bat」中的「用 Cmder 替换系统的命令行程序 cmd」。
  3. 移动位置后，ykdl/youtube-dl/you-get 这三个组件需要重新安装，重新安装前清理已安装的组件。
  4. ykdl 在有中文路径的情况下下载会出错，其他组件没问题，所以不建议放在中文目录下。

下载常见问题

开源项目找不到下载链接？

  • 所有开源项目打开后点击 releases 就是下载页面了，或者在项目主页网址后面+「/releases」进行访问。

什么，你找不到 releases？

  • 请按快捷键 Ctrl+F，输入releases，这个单词就会高亮显示了。

什么，你不知道下载哪个？

  • 别人命名不是乱命名的，Windows 系统就下名字中有 Windows 的，32 位系统下载 x86，64 位系统下载 x64，其他系统同理。
